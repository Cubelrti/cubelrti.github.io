# Prototype, multiply

让下面的代码运行

const a = [ 1, 2, 3, 4, 5]
a.multiply();
console.log(a) // 1, 2, 3, 4, 5, 1, 4, 9, 16, 25

Solution:

Array.prototype.multiply = function() {
    return this.concat(this.map(i => i*i))
}
